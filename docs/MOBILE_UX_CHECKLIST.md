# Mobile UX Test Checklist

Bu kontrol listesi canlı cihaz veya tarayıcı emülasyonu ile uygulanmalıdır.

## Ekranlar

- Login ekranı
- Gösterge Paneli
- Rapor Dizini
- ÖDR Veri Giriş Terminali
- Kanıt Arşivi
- Tablo Yönetimi
- Onay Akışı
- Ayarlar & Yedek

## Kontroller

- Sidebar metinleri taşmıyor.
- Rapor Dizini kartları dikeyde okunabilir.
- Tablo editörleri dar ekranda kullanıcıyı uyarıyor.
- Dosya yükleme butonu mobilde erişilebilir.
- Aç butonları doğru başlığa yönlendiriyor.
- Kaydetme sonrası güncel veri görünüyor.
- Modal/expander içerikleri ekran dışına taşmıyor.
- Export/indir butonları dokunmatik kullanımda yeterli büyüklükte.

## Kabul kriteri

Kritik iş akışları mobilde bozulmadan tamamlanabiliyorsa mobil destek "uygun" kabul edilir. Tablo düzenleme gibi geniş ekran gerektiren işlemlerde uygulama kullanıcıya açık uyarı göstermelidir.
